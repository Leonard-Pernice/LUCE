Welcome to LuceVM

Congratulations on setting up the LuceVM™. LuceVM provides a convenient development environment for the LUCE technical implementation. It abstracts away the complexity of the underlying libraries. (The author, Arno, at some point found himself in dependency hell while trying to set up the solidity compiler and all other components on his Macbook. So he decided to start afresh, build a clean development environment on this Ubuntu 16.04 server and expose the desired functionality to the host machine via port forwarding.) I am very happy with the result and hope it will prove useful for other users as well.
